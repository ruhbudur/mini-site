Kod parçacıkları

Sık lazım olan bazı küçük kodlar. Üstteki “Kopyala” butonuna tıklayıp doğrudan kendi projenine yapıştırabilirsin.

Python

1. 1 ile 100 arası rastgele sayı
import random

sayi = random.randint(1, 100)
print("Tuttuğum sayı:", sayi)

Basit tahmin oyunları veya testler için başlangıç kodu.

2. Dosyadan satır satır okuma
with open("veri.txt", "r", encoding="utf-8") as f:
    for satir in f:
        print(satir.strip())

Metin dosyalarındaki verileri satır satır işlemek için kullanılır.

3. Basit zamanlayıcı (geri sayım)
import time

saniye = 10
while saniye > 0:
    print(saniye)
    time.sleep(1)
    saniye -= 1

print("Süre doldu!")

Konsolda küçük geri sayım/timer yazmak için kullanılabilir.

JavaScript

1. Butona tıklayınca arka plan rengini değiştir
const colors = ["#0f172a", "#1d4ed8", "#16a34a", "#be123c"];

document.querySelector("button").addEventListener("click", () => {
  const secilen = colors[Math.floor(Math.random() * colors.length)];
  document.body.style.backgroundColor = secilen;
});

Basit animasyonlar veya tema değiştirme butonları için örnek kod.

2. Input alanındaki değeri canlı göster
const input = document.querySelector("#isim");
const cikti = document.querySelector("#merhaba");

input.addEventListener("input", () => {
  cikti.textContent = `Merhaba ${input.value || "?"}`;
});

Formlarda canlı önizleme için çok işe yarar.

C++

1. Dizideki en büyük eleman
#include <iostream>
#include <vector>
using namespace std;

int main() {
    vector<int> a = {3, 7, 2, 9, 4};
    int maks = a[0];
    for (int x : a) {
        if (x > maks) maks = x;
    }
    cout << "En büyük:" << maks << endl;
    return 0;
}

Dizilerle çalışırken en temel arama örneklerinden biri.
